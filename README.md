# Портфолио Веб-разработчика

Современный адаптивный сайт-портфолио, созданный с использованием передовых веб-технологий и лучших практик разработки.

## Особенности проекта

### Дизайн и UX
- **Адаптивный дизайн** с мобильного подхода (mobile-first)
- **Современный интерфейс** с плавными анимациями и переходами
- **Темная/светлая тема** с переключателем
- **Интуитивная навигация** с фиксированным хедером

### Технологии
- **HTML5** с семантической структурой
- **SCSS** с методологией БЭМ
- **CSS Grid** и **Flexbox** для макетов
- **JavaScript** (ES6+) для интерактивности
- **Normalize.css** для кроссбраузерности

### Доступность (a11y)
- Полная поддержка клавиатурной навигации
- Скринридеры и технологии assistive technology
- Контрастность ≥ 4.5:1
- Семантическая разметка
- Альтернативные тексты для изображений
- Валидные формы с метками

### Адаптивность
- **Mobile-first** подход
- Адаптивная сетка портфолио
- Отзывчивые компоненты
- Оптимизация для всех устройств
